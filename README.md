# Визуализация сумм Дарбу

Интерактивная визуализация верхних и нижних сумм Дарбу для функции на заданном интервале.

## Описание

Программа визуализирует приближение определенного интеграла с помощью верхних и нижних сумм Дарбу. При изменении количества разбиений с помощью слайдера, вы можете наблюдать, как суммы Дарбу сходятся к точному значению интеграла.

## Установка

1. Клонируйте репозиторий или скачайте файл `plotDarbuSums.py`

2. Установите необходимые зависимости:
```bash
pip install matplotlib numpy
```

## Запуск
```bash
python plotDarbuSums.py
```

После запуска откроется интерактивное окно с графиком функции и слайдером в нижней части.

## Управление
Слайдер "Шаг дискретизации": изменяет количество разбиений интервала (5-100)

При движении слайдера автоматически пересчитываются и перерисовываются прямоугольники Дарбу

В левом верхнем углу отображаются текущие значения сумм
